# Programming Portfolio

## Project: Count Lowercase 'a' Letters in a String (C) until '@'

### Description
A simple C program that counts the number of lowercase `'a'` letters in a user-provided string.  
The program stops counting when it encounters the `'@'` symbol.

### Purpose
This project demonstrates fundamental C programming skills, including:

- Working with arrays and character manipulation
- Using loops and conditional statements
- Creating and using functions with return values
- Handling basic user input safely

### How it works
1. The user is prompted to enter a string (up to 99 characters).  
2. The program counts the number of lowercase `'a'` letters until it reaches the `'@'` symbol.  
3. The total count is displayed to the user.

### Example
```
Enter a string (use '@' to stop): abcaa@
Number of 'a' letters: 3
```
